# Sinkd
A file management website that brings the benefits of storage to the cloud. Access your files anywhere, at any time.

# Features
- Account Creation
- Sortable Table
- Folders
- File/ Management:
    Uploading
    Renaming
    Deleting
    Viewing
    Sharing
- Account Management:
    Change Password
    Change Display Name
    Delete Account
    Sessions

# Technologies
- Bootstrap
- TableSorter
- jQuery
- AJAX
- MYSQL
- HTML, CSS, Javascript, PHP,
