# Sinkd
A file management website that brings the benefits of storage to the cloud. Access your files anywhere, at any time.

![Image](https://i.imgur.com/eYa1Lrq.png)

Features
- Account Creation
- Sortable Table
- Folders
- File/ Management:
    Uploading
    Renaming
    Deleting
    Viewing
    Sharing
- Account Management:
    Change Password
    Change Display Name
    Delete Account
